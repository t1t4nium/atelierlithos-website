# Informations projet

Les informations concernant le projet sont dans le fichier README.md

## Informations sensibles

- Le nom de variable d'une information sensible commence par SECRET_ (majuscule ou minuscule)
- Les valeurs se trouvent dans le fichier SECRETS.yml

### Très important
- Le fichier SECRETS.yml doit être ignoré/exclu par Git et Jekyll
- Les informations sensibles ne doivent JAMAIS apparaître dans le code source

### Exemples d'information sensible
- identifiant et mot de passe
- nom et prénom
- adresse postale
- adresse email
- numéro de téléphone
- raison sociale
- statut et SIRET/SIREN
- numéro carte bancaire et IBAN

## Modification projet

Mettre à jour le fichier README.md après toute modification de la structure du projet
